# 💫 Meta AI "Imagine Me" Prompt Composer

The ultimate open-source prompt builder for Meta AI's **"Imagine Me"** feature (used natively inside Instagram, WhatsApp, and Facebook). 
Easily generate perfectly structured image prompts, animation guide prompts, and ready-to-use social media captions—all through a beautifully designed, click-based interface.

[**🌐 View Live Demo**](https://alekhya-induri.github.io/Meta-AI-Prompt-Composer/)

---

## ✨ Features

- **Visual Prompt Building:** Select from dozens of curated identities, moods, outfits, settings, lighting, and camera angles. No more starting at a blank text box.
- **Live Preview:** See your exact prompt construct itself in real-time as you mix and match attributes.
- **Smart Captions:** Automatically generates highly relevant Instagram/social media captions and hashtags based on the exact keywords you selected.
- **Animation Prompts:** Creates secondary prompts specifically optimized for animating your AI-generated images.
- **Presets & History:** Save your favorite prompt combinations locally, and view a continuous history log of your 50 most recently copied prompts.
- **Export Options:** Export your prompts safely to `.txt` or `.json` formats.
- **Mobile Optimized:** A sleek, fully responsive interface that works flawlessly on your phone.

## 🛠️ Tech Stack

- **React 18** (UI rendering)
- **TypeScript** (Type safety)
- **Vite** (Rapid build tool & Dev server)
- **Tailwind CSS v4** (Styling & layout)
- **Lucide React** (Beautiful iconography)

## 🚀 Local Development Setup

Want to run this app locally on your own machine? It's super easy!

1. **Clone the repository**
   ```bash
   git clone https://github.com/alekhya-induri/Meta-AI-Prompt-Composer.git
   cd Meta-AI-Prompt-Composer
   ```

2. **Install dependencies**
   Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
   The app will compile instantly and be available at `http://localhost:3000`.

## 🤝 How to Contribute

Contributions, issues, and feature requests are highly welcome! If you want to add new prompt categories, fix bugs, or improve the UI, here is how you can safely and easily do it using your own branch:

1. **Fork the Project**  
   Click the "Fork" button at the top right of this repository to create your own copy on your account.

2. **Clone your fork locally:**
   ```bash
   git clone https://github.com/YOUR_GITHUB_USERNAME/Meta-AI-Prompt-Composer.git
   cd Meta-AI-Prompt-Composer
   ```

3. **Create a Feature Branch:**  
   Always create a new branch for your changes instead of working on the `main` branch.
   ```bash
   git checkout -b feature/MyAmazingFeature
   ```

4. **Commit your Changes:**  
   Make your code changes, then commit them with a clear message.
   ```bash
   git commit -m "Add some AmazingFeature"
   ```

5. **Push to the Branch:**
   ```bash
   git push origin feature/MyAmazingFeature
   ```

6. **Open a Pull Request!**  
   Go to the original repository on GitHub, and you'll see a green button to create a Pull Request from your new branch.
